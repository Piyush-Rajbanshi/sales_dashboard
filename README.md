📊 Sales Analytics Dashboard

A complete end-to-end data analytics project built using Python.
This project analyzes sales data, generates business insights, visualizes trends, and provides an interactive dashboard.

🚀 Features

Load and clean CSV sales data

Revenue calculation and KPI analysis

Monthly revenue trend analysis

Product and region performance insights

Static chart generation (PNG reports)

Interactive dashboard using Streamlit

🛠️ Tech Stack

Python

Pandas

Matplotlib

Streamlit

📈 Business Insights Generated

Total Revenue

Total Orders

Average Order Value

Top Performing Product

Best Performing Region

Monthly Revenue Trends

▶️ How to Run

Clone the repository

Create virtual environment

Install dependencies:

pip install -r requirements.txt

Run CLI analysis:

python analysis.py

Generate visual reports:

python visualize.py

Launch dashboard:

streamlit run dashboard.py
